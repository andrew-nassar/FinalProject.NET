# ⚖️ Lawyer Management Backend (ASP.NET Core API)

A complete backend system built with **ASP.NET Core** designed to manage lawyers, clients, cases, and appointments efficiently.  
This project focuses on providing a secure, scalable, and well-structured API for a modern law firm or legal management platform.

---

## 🚀 Features

- 👩‍⚖️ **Lawyer Management** – Create, update, and manage lawyer profiles with specialization and experience details.  
- 👨‍💼 **Client Management** – Register clients and link them to their assigned lawyer(s).  
- 📁 **Case Tracking** – Add, update, and monitor case progress with status and deadlines.  
- 📅 **Appointments System** – Schedule and manage meetings between lawyers and clients.  
- 🔒 **Authentication & Authorization** – Secure login and role-based access using JWT.  
- 💾 **Database Integration** – Built with **Entity Framework Core** and **SQL Server**.  
- 🌐 **RESTful API** – Follows clean architecture and industry best practices.
